FRE-VIC RC Wireless Trailer Lights System Dashboards installer

This installer change the Dashboard design to the RC TLS Tx, please follow the instructions below:

  1. Connect your RC TLS Tx device to a USB port
  2. Select the Dashboard you want to install
  3. Click "Connect" and then select the correct COM port. No device board found?
  4. The programming will take less than 3 minutes
  5. Restart the RC TLS Tx device
